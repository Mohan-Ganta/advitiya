Technical

Web Designing

It is an event of designing web page for the given theme. The event is organized in the CSE department labs. The time given for each participant is about 2-3 hours , in the given time they need to complete the web page and the theme will be given in the spot.

Blind Coding

Blind coding is same as coding but the participant is to execute the code without seeing it on the screen , which means after opening the compiler the monitor will be turned off by the respected coordinators after that question will be given to the participants.

Video Editing

The concept of video editing is to bring out the editing talent from the students. The theme of video editing is to edit a video on our college with their creative thoughts they can add background score and voice.

Innovative Thought

The main aim of this event is to bring out the problem solving innovative ideas from the students. For this event a problem is given for the participants and ask them how they can solve the problem based on their innovative thoughts the winner will be decided.

Memory Challenge

The event is to test the memory power of the students. In this event a picture is displayed for the participants for upto 15 seconds. After that questions will be asked based on that picture.
